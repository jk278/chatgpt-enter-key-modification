# chatgpt-enter-key-modification
一个 ChatGPT 回车处理脚本

## 它能做什么？

1. 修复 ChatGPT PC 端网页窗口变窄时的回车发送操作
2. 为 ChatGPT 移动端网页添加是否回车发送的选择按钮
